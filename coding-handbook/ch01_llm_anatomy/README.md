# Chapter 1: The Anatomy of an LLM — Code Lab

> 📖 **Book Chapter**: [Chapter 1 — The Anatomy of an LLM](../../book/chapter_1_anatomy.md)

## What You'll Build
- Scaled Dot-Product Attention from scratch in NumPy
- KV Cache memory calculator for any LLM architecture
- BPE tokenizer demonstration

## Files

| File | Description |
|------|-------------|
| `scaled_dot_product_attention.py` | Full implementation of masked self-attention |
| `kv_cache_calculator.py` | Calculate VRAM requirements for any model config |
| `bpe_tokenizer_demo.py` | Demonstrate how BPE tokenization affects agent performance |

## Run

```bash
python scaled_dot_product_attention.py
python kv_cache_calculator.py
python bpe_tokenizer_demo.py
```
